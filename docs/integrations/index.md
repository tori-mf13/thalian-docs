# Integration Guides

Step-by-step setup instructions for each platform Thalian supports. Select your integration below for prerequisites, credential creation, and connection steps.

---

## Identity & Access

- **[Okta](okta.md)** — API token setup for users, groups, MFA, and apps
- **[Microsoft Entra ID](microsoft-entra-id.md)** — OAuth or API setup for directory, sign-in logs, and conditional access
- **[Google Workspace](google-workspace.md)** — OAuth setup for users, groups, OAuth apps, and audit events
- **[JumpCloud](jumpcloud.md)** — API key setup for users, devices, and policies
- **[OneLogin](onelogin.md)** — Client credentials setup for users, apps, and roles

## Endpoint Management

- **[Microsoft Intune](microsoft-intune.md)** — OAuth or API setup for devices and compliance
- **[Jamf Pro](jamf-pro.md)** — API credentials setup for Mac/iOS devices and configurations
- **[Iru (formerly Kandji)](kandji.md)** — API token setup for Apple devices and blueprints
- **[Hexnode](hexnode.md)** — API key setup for cross-platform devices and policies

## Security

- **[CrowdStrike](crowdstrike.md)** — API credentials setup for endpoints, detections, and containment
- **[SentinelOne](sentinelone.md)** — API token setup for agents, threats, and device health

## Network

- **[Cisco Meraki](cisco-meraki.md)** — API key setup for network devices, clients, and VPN
- **[Auvik](auvik.md)** — API key setup for network devices, alerts, and topology

## ITSM

- **[Jira](jira.md)** — OAuth or API setup for issues, service requests, and alert delivery
- **[ServiceNow](servicenow.md)** — API credentials setup for incidents and CMDB
- **[Freshservice](freshservice.md)** — API key setup for tickets and assets
- **[Zendesk](zendesk.md)** — API token setup for tickets and organizations

## Communication

- **[Slack](slack.md)** — OAuth setup for finding alert delivery
- **[Microsoft Teams](microsoft-teams.md)** — OAuth or webhook setup for finding alert delivery
- **[Microsoft Outlook](microsoft-outlook.md)** — OAuth setup for mailbox monitoring and forwarding rule detection

## Outbound

- **[Webhooks](webhooks.md)** — Send findings to SIEMs, SOARs, custom dashboards, or any HTTPS endpoint

## Collaboration

- **[SharePoint](sharepoint.md)** — OAuth setup for sites, external sharing, and permissions
- **[Confluence](confluence.md)** — OAuth or API setup for spaces and content exposure
